*VARIBALE:
Una variable es un elemento a la que se le asigna un valor como un número(1,2,3...), una palabra('Bom dia', 'Henry'...etc), un caracter('h', 'H', '3'... etc) un valor booleano(true, false).
Este valor  puede ir varíando a lo largo del proyecto o del programa.
Ejemplos de variables:
var num = 0;
var esMascota= 'Perro';
var aprendoJs = true;
En estos ejemplos utilizamos la palabra recervada var la cual indica que estamos declarando una variable. Le precede el nombre de la variable (num, esMascota, aprendoJs)...es recomendable que cuando declaremos el nombre de una variable usemos una notación camel, es decir empezar con letras minuscula y para diferencias  otra palabra escribimos su primera letra en mayuscula y luego en minuscula.
Luego usamos el signo  = para asignale un valor determinado que puede ir cambiando a lo largo del programa como se mencionó anteriormete.
Cuando declaramos una variable y le establecemos un valor determinado estamos indicando la clase de tipo de dato a la que pertenece (Entero, Booleano, String, etc)  y el conjunto de operaciones posibles que podemos realizar con las varaibles declaradas.

*STRING: 
Es una clase de tipo de datos, la cual se define como cadena o un hilo de caracteres, deriva de char, el cuál es el tipo de dato que permite almacenar en una variable caracteres.
Ejemplos de String:
var nuevoString = 'Hola';
var nombre = 'Sol';
var apodo = 'Solchu';

Ejemplos de Char:
var nuevoChar = 'U';
var estoEsUnCaracter = '+';

Como podemos ver al declarar una variable String usamos la palabra recervada var, luego indicamos el nombre de la varaible, y le asigamos un valor el cuál para indicar que estamos hablando de un String(una cadena de caractéres) utilizamos las comillas simples (') .

*FUNCIÓN:
Una función es una estructura la cuál permite encerrar un bloque de código.
Las funciones son muy útiles a la hora de desarrollar un proyecto porque nos permite ahorrar lineas que se repiten en todo el código.
function nombreDeLaFuncion( parametroUno, parametroDos){
  Código de la función...
return;
}
Para declarar un función en JS usamos la plabra recervada function y luego el nombre de la función, abrimos paréntesis escribimos los argumento que valla a tomar esa función a la hora de ser llamada o invocada luego cerramos paréntesis y por ultimo abrimos llaves para escribir el código de la función.
Es importante resaltar que una función siempre es finalizada con un return el cuál devuelve o retórna los valores de la función. 
También cabe recordar que las varíables declaradas dentro de una función solo son posible acceder a ellas dentro de la función y no podemos asignarle un nuevo valor fuera de la función. 
*ARGUMENTOS DE UNA FUNCIÓN:
Los argumentos o también llamados parámentros  son valores que reciben las funciones al momento de ser llamadas a lo largo de código, las cuáles se manipulan dentro de la función para llegar a una respuesta.
*RETURN:
Es utilizado para finalizar la ejecución del código de la función y también permite especificar un valor para devolver cuando la función es llamada.
*DECLARACIONES IF:
Las declaciones if son estructura del control o flujo de control también llamdas en inglés statement. If es un bloque de código, una estrucura la cuál se utiliza para tomar desiciones en función a una condición.  Esta condición tiene que ser verdadera (true) para que se ejecute todo el código dentro de bloque de if. En caso de no cumplirse la condición (false) no se ejecuta ninguna instrucción dentro de la estructura y el programa sigue ejecutando el resto  de las instrucciones.
Ejemplo del flujo de control if:
if (condición ó expresión lógica){
código...
}

*VALORES BOOLEANOS:
Los valores booleanos son muy simples y son dos true o false, estos proviene de la lógica de Boole.
Ejemplo de valores booleanos:
var estoyFeliz = true;
var estaLloviendo = false;
